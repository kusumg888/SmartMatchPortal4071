# 🧠 Smart Match Lost and Found Portal

## 📖 About the Project

Smart Match Lost and Found Portal is a web-based application designed for college campuses to help students report and recover lost items efficiently. 
The system provides a centralized digital platform where users can post details about lost or found items, upload images, search existing posts, and connect with other users.
What makes this project unique is its AI-based text matching system, which automatically suggests possible matches between lost and found posts.

---

## 🎯 Objective

In many colleges, lost and found items are handled informally through notice boards or word of mouth, which makes recovery difficult and time consuming.
This project solves that problem by.
1. Creating a centralized platform
2. Reducing manual searching
3. Using AI to suggest possible matches automatically
4. Making communication easier between students

## 🧩 How the AI Matching Works (Concept)

1. User posts a lost item.
2. Another user posts a found item.
3. The system compares text descriptions using an AI-based similarity algorithm.
4. If similarity score crosses a threshold, the system suggests a match.
5. Users are notified about possible matches.
